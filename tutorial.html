1️⃣ Cara Copy Widget ke Project Lain

Untuk menggunakan widget ini di project lain, kamu hanya perlu 2 file:

1 Copy knob.js ke project kamu
2 Copy knob.css ke project kamu
3 Sertakan CDN Canvas Gauges di halaman HTML kamu
4 Buat container <div id="gauge-saya"></div>
5 Instantiate class KnobGauge di JavaScript
💡 Info: Kedua file (knob.js dan knob.css) sudah self-contained - tidak ada dependensi file lain dari project ini.

2️⃣ Setup Awal

Langkah 1: Sertakan Library Canvas Gauges

Widget ini membutuhkan library Canvas Gauges dari CDN. Tambahkan baris berikut di dalam tag <head>:

<script src="https://cdn.jsdelivr.net/npm/canvas-gauges@2.1.7/gauge.min.js"></script>

Langkah 2: Copy & Sertakan File Widget

Copy knob.js dan knob.css ke folder project kamu, lalu sertakan di HTML:

<!-- CSS Widget -->
<link rel="stylesheet" href="path/ke/knob.css">

<!-- JS Widget -->
<script src="path/ke/knob.js"></script>

Langkah 3: Buat Container

Buat elemen <div> sebagai tempat widget akan di-render:

<!-- Container untuk widget -->
<div id="my-gauge"></div>

Langkah 4: Inisialisasi Widget

Instantiate class KnobGauge dengan ID container dan opsi yang diinginkan:

<script>
    document.addEventListener('DOMContentLoaded', () => {
        const gauge = new KnobGauge('my-gauge', {
            minValue: 0,
            maxValue: 100,
            value: 50,
            width: 300,
            height: 300
        });

        // Set nilai baru
        gauge.setValue(75);

        // Dapatkan nilai saat ini
        const currentValue = gauge.getValue();
        console.log(currentValue); // 75

        // Event listener saat nilai berubah
        gauge.on('change', (data) => {
            console.log('Nilai berubah:', data.value);
        });
    });
</script>
⚠️ Penting: Pastikan Canvas Gauges sudah dimuat SEBELUM knob.js, karena KnobGauge membutuhkan class RadialGauge.

3️⃣ API Reference

Constructor: new KnobGauge(container, options)

Parameter Tipe Default Deskripsi
container string / HTMLElement wajib ID atau elemen DOM container
options.minValue number 0 Nilai minimum gauge
options.maxValue number 100 Nilai maksimum gauge
options.value number 0 Nilai awal gauge
options.width number 300 Lebar canvas gauge (px)
options.height number 300 Tinggi canvas gauge (px)
options.units string "" Satuan (contoh: "%", "°C", "km/j")
options.animationDuration number 1000 Durasi animasi dalam milidetik
options.animationRule string "linear" Jenis animasi (linear, el, bounce, dll)
options.highlights Array [...] Array warna segmen
options.showValueDisplay boolean true Tampilkan nilai numerik di tengah

Methods

Method Deskripsi
setValue(value, silent?) Set nilai gauge. silent=true untuk suppress event 'change'
getValue() Mengembalikan nilai gauge saat ini
on(event, callback) Daftarkan event listener. Event: 'change'
off(event, callback?) Hapus event listener
draw() Render ulang gauge
destroy() Bersihkan instance dan hapus dari DOM

4️⃣ Kustomisasi Tampilan

Warna Highlight

Atur warna segmen gauge dengan opsi highlights:

const gauge = new KnobGauge('my-gauge', {
    highlights: [
        { from: 0, to: 40, color: "#22c55e" },   // Hijau
        { from: 40, to: 70, color: "#eab308" },  // Kuning
        { from: 70, to: 100, color: "#ef4444" }  // Merah
    ]
});

Variasi Ukuran via CSS

Widget mendukung variasi ukuran melalui class CSS pada container:

<!-- Ukuran kecil -->
<div id="my-gauge" class="knob-small"></div>

<!-- Ukuran besar -->
<div id="my-gauge" class="knob-large"></div>

<!-- Tema terang -->
<div id="my-gauge" class="knob-light"></div>

Opsi Tambahan Warna

Kustomisasi warna lebih detail:

const gauge = new KnobGauge('my-gauge', {
    colorPlate: "#1e293b",        // Warna latar gauge
    colorMajorTicks: "#94a3b8",  // Warna tick utama
    colorMinorTicks: "#475569",  // Warna tick minor
    colorNumbers: "#f1f5f9",     // Warna angka
    colorNeedle: "#f97316",      // Warna jarum
    colorNeedleEnd: "#ea580c",   // Warna ujung jarum
    needleType: "arrow",         // Tipe jarum: arrow, line
    needleWidth: 4               // Lebar jarum
});

5️⃣ Contoh Lengkap

Berikut contoh HTML lengkap dari awal sampai akhir untuk implementasi widget:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Project Saya - Widget Knob</title>

    <!-- 1. Canvas Gauges CDN -->
    <script src="https://cdn.jsdelivr.net/npm/canvas-gauges@2.1.7/gauge.min.js"></script>

    <!-- 2. Widget Knob (copy dari project) -->
    <link rel="stylesheet" href="asset/widget_knob/knob.css">
    <script src="asset/widget_knob/knob.js"></script>
</head>
<body>
    <!-- 3. Container -->
    <div id="suhu-gauge"></div>
    <p>Suhu: <span id="suhu-value">25</span>°C</p>

    <script>
        document.addEventListener('DOMContentLoaded', () => {
            // 4. Buat instance
            const gauge = new KnobGauge('suhu-gauge', {
                minValue: 0,
                maxValue: 50,
                value: 25,
                width: 250,
                height: 250,
                units: '°C',
                animationDuration: 500,
                highlights: [
                    { from: 0, to: 20, color: '#3b82f6' },
                    { from: 20, to: 35, color: '#22c55e' },
                    { from: 35, to: 50, color: '#ef4444' }
                ]
            });

            // 5. Gunakan method
            gauge.setValue(32);

            gauge.on('change', (data) => {
                document.getElementById('suhu-value')
                    .textContent = Math.round(data.value);
            });
        });
    </script>
</body>
</html>
